# 2nd\&2 Sports — Website

Official website for **2nd\&2**, an elite sports development business offering athlete training camps, competitive tournaments, branded merchandise, and media/interview content.

## About

2nd\&2 is built on the belief that every athlete deserves a shot. The name comes from the football down "2nd and 2" — a situation where you're set up for success. The business provides:

* **Camps** — Elite skills training, speed \& agility, QB academy
* **Tournaments** — 7-on-7 series, regional showcases, college exposure events
* **Merchandise** — Performance apparel, lifestyle gear, headwear
* **Interviews** — Athlete spotlights, coach breakdowns, recruit diaries

## Deployment Steps (Summary)

1. Rename main HTML file to `index.html`
2. Push to GitHub repository
3. Enable GitHub Pages: Settings → Pages → Deploy from branch (main, root)
4. Add custom domain in GitHub Pages settings
5. Update DNS at GoDaddy:

   * 4x A records pointing to GitHub Pages IPs (185.199.108–111.153)
   * CNAME record for `www` pointing to `yourusername.github.io`
6. Enable "Enforce HTTPS" once DNS propagates
